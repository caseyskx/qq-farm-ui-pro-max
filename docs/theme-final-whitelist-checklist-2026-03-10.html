你提到的“iframe 主题桥接定义，必要保留”指的就是 农场工具页（`FarmTools.vue`）向内部 iframe 注入主题变量。已做同步优化，改为优先使用语义品牌变量桥接（`--ui-brand-*`）。

最终必要语义色白名单

类别允许保留（示例）说明
全局语义 Token --ui-bg-*, --ui-text-*, --ui-border-*, --ui-brand-* 全站主题主来源，允许保留且鼓励使用。
业务语义状态色 --ui-status-success|warning|danger|info 与其 *-soft 用于成功/警告/错误/信息语义，不属于“杂色”。
动态混色表达 color-mix(in srgb, ...) 用于透明度、遮罩、弱底色，属于规范化策略。
FarmTools iframe 桥接 --farm-brand-*（由 --ui-brand-* 映射） 同源 iframe 内覆盖 Tailwind 类所需，属于必要保留。
表单报错语义色 text-red-600 / dark:text-red-400 `UserInfoCard` 的续费/改密报错提示，属于必要错误语义，不纳入杂色。

自动巡检规则（已落地）

  • text-white/95~70 自动增强到高可读文本色。
  • text-white/60~40 自动提升对比，避免暗背景发灰。
  • 弱对比圆角徽章（rounded-full + bg-black/bg-white + text-gray-300/400/500）自动回正到可读文本色。
  • 弱对比圆角块（rounded-* + bg-black/bg-white + text-gray-300/400）自动回正到可读文本色。
