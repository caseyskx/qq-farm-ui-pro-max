# REPORT_复刻QQ农场站点

## 1. 交付说明
按照“最简单的方式完美 100% 复刻”原则，本项目摒弃了任何复杂的 Java 或 Node.js 框架，采用**纯静态文件 + 极简 Python 原生代理**的本地运行方案。

所有静态资源（包含图片、CSS、JS 等 100+ 个文件）均已完整离线到本地 `levels_replica` 目录，并修正了原本的绝对路径为本地相对路径。

针对网页前端使用的 `fetch('/api/...')` 动态查询接口，专门编写了一个总计不到 40 行的 Python 原生反向代理脚本 (`server.py`)，无需安装任何第三方库，在解决跨域问题的同时，将接口无缝衔接到原本的服务端，从而在本地实现了 100% 的功能复刻。

## 2. 最终测试结果
- **页面访问**：首页、经验计算器、氪金计算器、图鉴等各个页面全部成功渲染，图片加载正常（包含 TailwindCSS 样式还原）。
- **动态交互**：氪金计算器等页面的 Ajax 数据拉取通过 `server.py` 转发测试状态 `HTTP 200`，工作流闭环完美打通。

## 3. 操作指引（如何运行）
### 第一步：打开终端并进入目录
```bash
cd /Users/smdk/Downloads/nc/levels_replica
```

### 第二步：启动极简本地服务
只需执行以下自带的 Python 脚本，就会自动开启静态资源与 API 接口代理（默认端口 8000）：
```bash
python3 server.py
```

### 第三步：浏览器访问
打开浏览器，访问以下地址即可体验 100% 复刻的站点：
- [http://localhost:8000/](http://localhost:8000/) 

> 提示：如果想直接访问氪金计算器，可打开 `http://localhost:8000/levels`
> 停止服务在终端按 `Ctrl + C` 即可。
